# Task Pro – Micro Job Platform

Task Pro is a micro-job web application where users can post small online tasks and other users can complete those tasks. The platform connects task creators and workers, making it easy to manage and complete small digital jobs.

## Live Website

https://task-pro-lake.vercel.app/

## Features

- **Task Posting System**  
  Users can create and publish micro tasks with details such as title, description, budget, and deadline.

- **Browse Available Tasks**  
  Workers can explore available tasks on the platform and choose tasks they want to complete.

- **Task Submission & Completion**  
  Users can submit proof after completing a task, allowing task creators to review the submission.

- **User Authentication**  
  Secure authentication system so users can sign up, log in, and manage their tasks.

- **Task Management Dashboard**  
  Users can track their posted tasks and completed tasks from a personal dashboard.

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/aminul-islam36/task-pro.git
```

Go to the project directory:

```bash
cd task-pro
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the app in your browser:

```
http://localhost:3000
```

## Project Purpose

The goal of this project is to build a simple micro-job marketplace where users can post tasks and others can complete them. It demonstrates full-stack development using the MERN ecosystem with Next.js.
